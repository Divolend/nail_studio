# Студия маникюра — веб-приложение

Курсовой проект по дисциплине «Разработка веб-приложений».
Django + Django Templates + Bootstrap 5 + SQLite.

## Установка и запуск (macOS)

1. Создайте и активируйте виртуальное окружение:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. Примените миграции (создастся база данных):
   ```bash
   python manage.py migrate
   ```

4. Создайте администратора (суперпользователя):
   ```bash
   python manage.py createsuperuser
   ```

5. Запустите сервер разработки:
   ```bash
   python manage.py runserver
   ```

6. Откройте в браузере:
   - Сайт: http://127.0.0.1:8000/
   - Админка: http://127.0.0.1:8000/admin/

## Структура проекта

- `accounts/` — пользователи (роли: клиент, мастер, администратор), профили мастеров
- `catalog/` — услуги студии, портфолио мастеров
- `booking/` — расписание (слоты) и записи на процедуры
- `subscriptions/` — типы абонементов и купленные абонементы
- `templates/` — HTML-шаблоны (Bootstrap 5)
- `media/` — загруженные изображения (портфолио, референсы)

## Создание мастера (для администратора)

Мастеров создаёт администратор через админку:
1. Раздел «Пользователи» → «Добавить»
2. Введите логин и **временный пароль**
3. На странице редактирования: роль = «Мастер», заполните ФИО,
   включите галочку «Требуется смена пароля»
4. Сообщите мастеру логин и временный пароль

При первом входе мастер будет обязан сменить пароль на свой —
после этого администратор пароль больше не знает (хранится в виде хэша).
Профиль мастера создаётся автоматически.
